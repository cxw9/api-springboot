API - 随机图片API

图片API基本调用格式：

javascript:; 
https://api.xingw.top/img/test（测试图片）

https://api.xingw.top/img/acg（动漫）

文字API基本调用格式：

https://api.xingw.top/yan/yiyan（一言语录）

https://api.xingw.top/yan/tgrj（舔狗日记）

参数：

type=json

JSON调用格式

图片列：https://api.xingw.top/api?return=json
文字列：https://api.xingw.top/yiyan?code=js

JSON数据

{
    "code":"200" #图片状态码
    "acgurl":"https:\/\/ws1.sinaimg.cn\/large\/0072Vf1pgy1foxkjcneg1j31kw0w0hbs.jpg" #图片地址
    "width":"2048" #图片宽
    "height":"1152" #图片高
}
